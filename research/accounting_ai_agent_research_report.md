### 5.2 핵심 에이전트 정의

#### Agent 1: 홈택스 자동화 에이전트
| 항목 | 내용 |
|-----|------|
| **역할** | 홈택스 데이터 수집 및 신고 자동화 |
| **입력** | 수임처 정보, 인증서, 신고 기간 |
| **출력** | 세금계산서 데이터, 카드 사용내역, 신고서 제출 |
| **연동** | 홈택스 API, 틸코블렛/바로빌 |
| **기술** | Playwright/Selenium (스크래핑), API 연동 |

**주요 기능**:
- 매입/매출 세금계산서 자동 수집
- 사업용 카드/계좌 내역 수집
- 민원증명 자동 발급
- 부가세 예정신고 자동화

---

#### Agent 2: 증빙 분류 에이전트
| 항목 | 내용 |
|-----|------|
| **역할** | 세금계산서, 카드 내역 자동 분류 및 분개 |
| **입력** | 원시 증빙 데이터 (PDF, 이미지, API 데이터) |
| **출력** | 분류된 거래, 자동 분개 |
| **기술** | OCR (Tesseract/Claude Vision), LLM 분류 |

**분류 체계**:
- 매입세액 공제/불공제 분류
- 계정과목 자동 매핑
- 거래처 자동 인식
- 중복 거래 탐지

---

#### Agent 3: 세무조정 보조 에이전트
| 항목 | 내용 |
|-----|------|
| **역할** | 법인세/소득세 세무조정 보조 |
| **입력** | 재무제표, 전기 세무조정 데이터 |
| **출력** | 세무조정 제안, 서식 자동 작성 |
| **기술** | RAG (세법 지식베이스), LLM |

**주요 기능**:
- 세무조정 항목 자동 탐지
- 세액공제/감면 요건 검토
- 누락 가능성 알림
- 전년 대비 분석

---

#### Agent 4: 컴플라이언스 에이전트
| 항목 | 내용 |
|-----|------|
| **역할** | 신고 기한 관리, 이상 탐지 |
| **입력** | 신고 일정, 거래 데이터 |
| **출력** | 알림, 이상 거래 리포트 |
| **기술** | 규칙 기반 + ML 이상탐지 |

**주요 기능**:
- 신고 기한 자동 알림
- 미납 세금 알림
- 비정상 거래 패턴 탐지
- 세무조사 리스크 평가

---

#### Agent 5: 보고서 생성 에이전트
| 항목 | 내용 |
|-----|------|
| **역할** | 고객용 보고서 자동 생성 |
| **입력** | 재무 데이터, 분석 결과 |
| **출력** | PDF/HTML 보고서, 이메일/카카오톡 발송 |
| **기술** | 템플릿 엔진, LLM 서술 |

**보고서 유형**:
- 월간 경영분석 보고서
- 부가세 신고 결과 보고서
- 세무 컨설팅 보고서
- 세액 시뮬레이션 보고서
